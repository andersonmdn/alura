## Rodando o Projeto
1. Intalar as dependencias:
	1. Yarn
	```
	yarn install
	```
	2. NPM
	```
	npm install
	```
2. Gerar TS para JS
```
yarn start
```

3. Rodar o arquivo **index.html** com o extensão **Live Server** no **VSCode**
